## 使用教程

#### 掉线通知
1. 文件在biscuit-plugin/apps下
2. 修改里面的相关材料(stmp服务器/用户名/密码/接收人邮件)

#### 简单使用
请对机器人发送`#饼干帮助`查看帮助

## 安装教程

请在机器人根目录输入

```
git clone --depth=1 https://gitee.com/Yummy-cookie/biscuit-plugin.git ./plugins/biscuit-plugin
```

## 安装依赖

```
pnpm install -p
```
两种安装方式
```
pnpm install --filter=biscuit-plugin
```
