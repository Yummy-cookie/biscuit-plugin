#### 安装教程

请在机器人根目录输入

```
git clone --depth=1 https://gitee.com/Yummy-cookie/biscuit-plugin.git ./plugins/biscuit-plugin
```

#### 安装依赖

```
pnpm install -p
```
两种安装方式
```
pnpm install --filter=biscuit-plugin
```
